# SOP: User Onboarding & Identity Verification
**Scope:** New hires, contractors, and interns at Milot Motors.

## Phase 1: Identity Verification
1.  Review Government ID (Driverâ€™s License/Passport).
2.  Log the employee's legal name and personal (emergency) email.

## Phase 2: Account Provisioning
1.  Create GitHub/Cloud account using the `firstname.lastname@milotmotors.com` format.
2.  Assign to the "Shop Tech" or "Admin" group based on the **Least Privilege** principle.
3.  **Prohibit** sharing of logins; every user must have their own "Digital Footprint."

## Phase 3: The Security Handshake
1.  User must login and immediately change their temporary password.
2.  User must enroll in **MFA** (Multi-Factor Authentication).
3.  User must sign the **Acceptable Use Policy (AUP)**.
